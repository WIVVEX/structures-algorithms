 Очередность создания методов LinkedList
1. ФУНДАМЕНТ
__init__() - конструктор

is_empty() - проверка пустоты

__str__() - строковое представление

2. БАЗОВОЕ ДОБАВЛЕНИЕ
append() - добавление в конец

prepend() - добавление в начало

3. ЧТЕНИЕ И ПОИСК
get() - получение по индексу

find() - поиск индекса по значению

contains() - проверка наличия

4. ПРОСТОЕ УДАЛЕНИЕ
delete_first() - удаление первого

delete_last() - удаление последнего

5. СЛОЖНОЕ УДАЛЕНИЕ
delete_at() - удаление по индексу

delete() - удаление по значению

6. ПРОДВИНУТЫЕ ОПЕРАЦИИ
insert() - вставка по индексу

reverse() - разворот списка

clear() - очистка списка

7. ДОПОЛНИТЕЛЬНЫЕ МЕТОДЫ
to_list() - преобразование в список

__iter__() - итератор
